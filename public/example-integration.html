Tree Diagram Widget Integration Examples

This page demonstrates how to integrate the Direct Sales Tree Diagram widget into your website.

Method 1: JavaScript Widget

Include the widget script and initialize with JavaScript.

Instructions: Enter a member ID or wallet address to load the tree diagram. 
// Include the widget script
<script src="https://yourdomain.com/widget.js"></script>

// Initialize the widget
const treeWidget = TreeWidget.init({
  container: 'tree-container-1',
  apiUrl: 'https://yourdomain.com/api',
  memberId: 1,
  maxDepth: 3,
  showControls: true,
  showTooltips: true,
  onNodeClick: (nodeId, nodeData) => {
    console.log('Node clicked:', nodeId, nodeData);
  }
});

Method 2: Iframe Embedding

Embed the tree diagram using an iframe.

<iframe 
  src="https://yourdomain.com/embed.html?memberId=1&maxDepth=3" 
  width="100%" 
  height="400px"
  frameborder="0">
</iframe>

Method 3: API Integration

Fetch data from the API and use your own visualization.

Click "Fetch Tree Data" to see API response...
// Fetch tree data
fetch('https://yourdomain.com/api/tree/1?maxDepth=3')
  .then(response => response.json())
  .then(data => {
    console.log('Tree data:', data);
    // Use your own visualization library
  });

// Search member
fetch('https://yourdomain.com/api/members/wallet/0x1234...')
  .then(response => response.json())
  .then(member => {
    console.log('Member:', member);
  });
